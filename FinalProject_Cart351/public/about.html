About Vocal Tag


The purpose of this website is to create an explorable networked-space that allows users to create and share their personal audio identity such as Vocal Tags. This concept is defined as a unique and personal sound, typically a few seconds long, characteristic of an individual.


If asked to describe oneself using only 3 words, in a relatively short amount of time, most people would succeed. However, if they are asked to describe themselves using only 3 seconds of sound, I believe most would find it more challenging.
